# Orange Quality Data Analysis (EDA)

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an Orange Quality dataset to understand the relationships between different physical and chemical characteristics of oranges and their quality rating. The analysis helps identify patterns and factors that may influence orange quality.

## Dataset Features

The dataset includes the following features:

* Size (cm)
* Weight (g)
* Brix (Sweetness)
* pH (Acidity)
* Softness (1–5)
* HarvestTime (days)
* Ripeness (1–5)
* Quality (1–5)

## Objectives

* Explore the dataset and understand feature distributions
* Analyze relationships between orange characteristics and quality
* Identify patterns using data visualization techniques

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Visualization

The following visualizations were used in this analysis:

* **Count Plot** – Shows the distribution of orange quality ratings
* **Scatter Plots** – Shows relationships between features and orange quality
* **Correlation Heatmap** – Displays correlation between numerical features

## Key Insights

* Visualization helps identify patterns and understand important factors affecting orange quality.
 
* Size (cm)
Size increase ayina quality score sometimes same ga untundi. Kabatti size alone orange quality ni strong ga decide cheyyadu.

 * Weight (g)
Weight change ayina quality lo clear pattern kanipinchakapovachu. Idi weight quality ki strong factor kaakapovachu ani suggest chestundi.
 
 * Brix (Sweetness)
Sweetness ekkuva unna oranges lo quality score kuda ekkuva undachu. Idi sweetness quality ni influence cheyyachu ani indicate chestundi.

 * pH (Acidity)
Acidity level change ayina quality lo konni variations kanipinchachu. Moderate acidity better quality ki sambandham undachu.

 * Softness (1-5)
Softness level change ayite quality score kuda change ayye chance undi. Proper softness unna oranges better quality indicate cheyyachu.

 * HarvestTime (days)
Harvest time different ga unna quality lo konni differences kanipinchachu. Correct harvest time better orange quality ki help cheyyachu.

 * Ripeness (1-5)
Ripeness increase ayite quality score kuda increase ayye tendency undachu. Fully ripened oranges generally higher quality ni indicate chestayi.

## Conclusion

Exploratory Data Analysis helped in understanding the dataset and identifying potential relationships between features and orange quality. Visualizations provided meaningful insights that help explain how different characteristics may influence the quality rating of oranges.
